📋 Pipeline (12 Steps)
  1. Unicode Normalization: Chuẩn hóa dấu tiếng Việt
  2. HTML/URL Removal: Xóa rác kỹ thuật
  3. Hashtag Removal: Xóa spam hashtag
  4. NER Masking: Ẩn danh tên người với whitelist
  5. Lowercase: Chuẩn hóa chữ thường
  6. Emoji → Tags: Chuyển emoji thành sentiment tags
  7. Text Emoticons: Xóa :)), =)), :D
  8. English Insults: Gắn tags cho từ tiếng Anh
  9. Bypass Patterns: n.g.u → ngu, đ-m → đm
  10. Leetspeak: ngu4 → ngua, ch3t → chết
  11. Repeated Chars: nguuuu → ngu <intense>
  12. Teencode: ko → không (với Intensity Preservation)
